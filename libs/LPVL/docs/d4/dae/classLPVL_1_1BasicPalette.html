Подробное описание

Простейшая палитра цветов.

Класс предоставляет набор цветов с сеттерами, геттерами и сигналами. Доступны следующие цвета:

  • цвет заднего плана
  • цвет переднего плана
  • промежуточный цвет
  • второй промежуточный цвет
  • акцентный цвет
  • цвет предупреждений
  • цвет ошибок
  • цвет оповещений

Класс зарегистрирован в мета-системе и доступен для встраивания в типы QML.
