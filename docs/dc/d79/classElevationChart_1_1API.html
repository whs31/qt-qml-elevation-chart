Подробное описание

Публичный интерфейс (C++) для виджета профиля высот.

Класс-синглтон. Предоставляет возможность вызывать функции профиля высот из C++ - кода. Для работы необходимо задать объект, который будет использоваться в C++ - см. setSource.

Заметки
Доступен как singleton в QML.
  • Модуль: ElevationChartWidget 3.0
  • Компонент: ElevationChartCXXAPI 3.0

◆ setSource()

void setSource ( ElevationChart::ChartItem *  item)

Задает объект, к которому будут направлены обращения из C++ кода в QML.

Пример:

ElevationChartImpl
{
id: impl
// ...
}
Component.onCompleted: ElevationChartCXXAPI.setSource(impl)

В случае успешного выполнения команды в консоль будет выведено сообщение, например:

<elevation-chart> Source set for C++ wrapper: ElevationChart::ChartItem(0x5607f4e241b0, parent=0x5607f4def790, geometry=0,0 1280x800)
ElevationChart::ChartItem
Definition elevationchart.h:15
Заметки
Эта функция может быть вызвана из QML через мета-объектную систему Qt.
Предупреждения
Если объект не будет задан, поведение виджета не определено. Объект необходимо задать до того, как будет вызвана любая другая функция API (кроме valid).
Аргументы
item- указатель на объект ChartItem.
