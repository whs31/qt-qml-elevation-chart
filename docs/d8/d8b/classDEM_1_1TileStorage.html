Подробное описание

Хранилище тайлов профилей высот.

Хранит, загружает и предоставляет доступ к набору тайлов высотных карт из указанной папки.

Является синглтоном. Класс - потокобезопасен.

Заметки
По умолчанию папка, в которой будет производиться поиск тайлов, равна пути к текущему исполняемому файлу приложения + /elevations.
